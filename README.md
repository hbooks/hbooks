# hb00ks – Author Website 

---

## ✨ Features

- **Homepage** – Animated hero section, latest news preview, and book spotlight.
- **About** – Author bio and publisher information with a clean, professional layout.
- **Books** – Detailed book page with synopsis, purchase links, and a "praise" snippet.
- **News** – Dynamic news posts with working like buttons (powered by Supabase).
- **Reviews** – Public review submission and display of approved reviews.
- **Contact** – Contact form that stores messages in the database (email notification coming soon).
- **Support** – Links to Buy Me a Coffee, Ko‑fi, and PayPal donation buttons.
- **Library** – Showcase of published and upcoming books.
- **Admin Portal** – Hidden admin area (triggered by triple‑clicking the logo) to manage books, news, and upcoming books.

---

## 🛠️ Tech Stack

- **Frontend:** React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend & Database:**  (PostgreSQL)
- **Hosting:** Cloudflare Pages
- **Icons:** Lucide React, Font Awesome
- **Package Manager:** npm

### Prerequisites
